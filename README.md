# # 115年度青年就業達人班 - 線上回饋系統

本儲存庫包含兩套問卷系統，所有資料皆匯流至：
[115年度_青達計畫_管理系統](https://docs.google.com/spreadsheets/d/1GYkFvQPXh4z_J2sqhbn2qPHrOdAMz5rZ9R3f4mdgoTs/edit)

## 系統版本說明

### 1. 青達班 (Project A)
* **適用對象**：一般的青年就業達人講座。
* **資料去向**：試算表中的 **「青達班」** 分頁。
* **特點**：欄位最精簡，適合手機快速填寫。

### 2. 十五堂課 (Project B)
* **適用對象**：包含參訪、15堂必修課、特定達人講座。
* **資料去向**：根據類別自動進入 **「達人講座」**、**「企業參訪」** 或 **「課程」** 分頁。
* **特點**：多步驟導引，含學制選擇，邏輯最完整。

## 開發者與視覺規範
* **視覺風格**：全系統採用 32px 大圓角、藍色裝飾條 (#2563eb)。
* **防呆機制**：所有聯絡資訊欄位均設為勾選後彈出，並具備標題列自癒功能。
